# Lughati — デザイン要件

## カラーパレット（桜色テーマ）

### CSS変数
```css
:root {
  /* プライマリー（桜色） */
  --color-primary: #E8567F;
  --color-primary-light: #FFF0F3;
  --color-primary-dark: #C4395F;

  /* アクセント（金色 / 星マーク） */
  --color-accent: #D4A853;
  --color-accent-light: #FDF6E3;
  --color-accent-dark: #B8903D;

  /* 背景 */
  --color-bg-page: #FFF7F9;       /* ページ背景（桜白） */
  --color-bg-card: #FFFFFF;        /* カード面（白） */

  /* テキスト */
  --color-text-primary: #1F1F23;   /* メインテキスト */
  --color-text-secondary: #8C8C96; /* サブテキスト */

  /* セマンティック（意味別） */
  --color-success: #2BA877;        /* 成功 */
  --color-danger: #D94452;         /* 危険・削除 */
  --color-danger-light: #FEF2F2;   /* 危険の薄い版 */

  /* ボーダー */
  --color-border: #F0E8EB;
}
```

### 使い方ルール
- ページ背景: 常に `--color-bg-page` (#FFF7F9)、純白は使わない
- カード面: `--color-bg-card` (#FFFFFF) + 薄い影
- プライマリーボタン: `--color-primary` 背景 + 白文字
- セカンダリーボタン: `--color-primary-light` 背景 + `--color-primary` 文字
- 星マーク（オン）: `--color-accent` (#D4A853)
- 星マーク（オフ）: `--color-border` (#F0E8EB)
- 削除・危険操作: `--color-danger`
- 成功フィードバック: `--color-success`

## タイポグラフィ（文字）

### フォントファミリー
```css
:root {
  --font-arabic: 'Noto Naskh Arabic', serif;
  --font-ui: 'Inter', system-ui, sans-serif;
}
```

- Google Fonts から読み込み: `Noto Naskh Arabic:wght@400;700` と `Inter:wght@400;500;600`
- アラビア語テキスト: 常に `--font-arabic`
- 英語/UIテキスト: 常に `--font-ui`

### フォントサイズ（カード内）
| 要素 | サイズ | 太さ | スタイル |
|------|--------|------|---------|
| アラビア語（カード） | 32px | 700（太字） | RTL、`--font-arabic` |
| 英語（カード） | 22px | 400 | LTR、`--font-ui` |
| ローマ字 | 13px | 400 | イタリック、`--color-text-secondary` |
| カード番号 | 13px | 500 | `--color-text-secondary` |

### フォントサイズ（UI）
| 要素 | サイズ | 太さ |
|------|--------|------|
| ページタイトル | 24px | 600 |
| セクション見出し | 18px | 600 |
| 本文 | 16px | 400 |
| ボタン文字 | 16px | 500 |
| ラベル・キャプション | 14px | 400 |
| 小さい文字・ヒント | 12px | 400 |

## レイアウト

### スペーシング（Tailwind）
- xs: 4px (p-1)
- sm: 8px (p-2)
- md: 16px (p-4)
- lg: 24px (p-6)
- xl: 32px (p-8)
- 2xl: 48px (p-12)

### 角丸
- ボタン: 12px (rounded-xl)
- カード: 16px (rounded-2xl)
- 入力フィールド: 12px (rounded-xl)
- ピル・バッジ: 9999px (rounded-full)

### 影
- カード: `0 1px 3px rgba(0, 0, 0, 0.06)`
- カード（ホバー時）: `0 4px 12px rgba(0, 0, 0, 0.08)`
- モーダル背景: `rgba(0, 0, 0, 0.4)`

## コンポーネント

### ボタンスタイル
```
プライマリー: bg-[#E8567F] text-white rounded-xl px-6 py-3 font-medium
セカンダリー: bg-[#FFF0F3] text-[#E8567F] rounded-xl px-6 py-3 font-medium
危険:        bg-[#FEF2F2] text-[#D94452] rounded-xl px-6 py-3 font-medium
ゴースト:    bg-transparent text-[#8C8C96] px-4 py-2
```

### カードコンポーネント
- 背景: 白 (#FFFFFF)
- 角丸: 16px
- 影: `0 1px 3px rgba(0, 0, 0, 0.06)`
- パディング: 24px
- ホバー時: 影が `0 4px 12px rgba(0, 0, 0, 0.08)` に増加

### 入力フィールド
- ボーダー: 1px solid `--color-border`
- 角丸: 12px
- パディング: 12px 16px
- フォーカス時: ボーダーが `--color-primary` に変化
- プレースホルダー: `--color-text-secondary`

### 星マークアイコン
- オフ: 輪郭のみ、色は `--color-border`
- オン: 塗りつぶし、色は `--color-accent` (#D4A853)
- サイズ: カード内24px、リスト内20px
- タップ領域: 最低44x44px

## 画面

### 1. ログイン / サインアップ
- 桜白の背景に中央寄せのカード
- 上部にアプリ名「Lughati」+ アラビア語「لُغَتِي」
- メール＋パスワード入力欄
- プライマリーボタン:「サインアップ」または「ログイン」
- 切り替えリンク:「アカウントをお持ちですか？」/「アカウント作成」
- ゴーストボタン:「ゲストとして続ける」

### 2. デッキ一覧（ホーム）
- 上部:「マイデッキ」見出し +「+」作成ボタン
- デッキカードのグリッド（モバイル1列、タブレット以上2列）
- 各デッキカード: タイトル、カード枚数、最終学習日
- 空の状態: イラスト +「最初のデッキを作成」ボタン

### 3. デッキ詳細
- ヘッダー: デッキタイトル + 編集/削除操作
- 「カード追加」と「CSVインポート」ボタン
- カード一覧（アラビア語 + 英語のプレビュー）
- 各カード行に星マークと編集/削除

### 4. カード編集
- モーダルまたは全画面フォーム
- 入力欄: アラビア語（RTL入力）、英語、ローマ字（任意）
- 保存/キャンセルボタン

### 5. CSVインポート
- ファイルアップロード領域（ドラッグ&ドロップ + クリック）
- 列マッピングのプレビュー（最初の5行）
- 確認/キャンセルボタン
- インポート後の結果サマリー

### 6. 学習モード
- 全画面カード表示
- 裏返しアニメーション（タップで裏返し）
- ナビゲーション: スワイプまたは矢印ボタン
- 上部バー: 戻るボタン、進捗「3/25」、星マーク
- 下部バー: シャッフル切替、表示モード切替
- 表示モード:
  - 通常: タップで裏返し（表→裏）
  - 両面表示: 上にアラビア語、下に英語（裏返し不要）
- スピーカーアイコンでTTS音声再生

### 7. 設定
- 音声: 女性（デフォルト）/ 男性 切替
- 自動再生: オン/オフ
- ローマ字表示: オン/オフ
- アカウント: ログアウト、アカウント削除
